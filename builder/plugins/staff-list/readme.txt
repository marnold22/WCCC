== Description ==

**Staff List** - staff directory of employees, faculty or staff members.

= Live Previews =

[http://abcfolio.com/wordpress-plugin-staff-list-live-previews/](http://abcfolio.com/wordpress-plugin-staff-list-live-previews/).

= Features =

* Responsive layout.
* Single page option.
* Easy to add to any page or post - just insert a shortcode.
* Custom templates. You can add any field in any order.
* Custom styles to create the uniform look and feel that match your theme.
* [Full list of features.](http://abcfolio.com/wordpress-plugin-staff-list-get-plugin/)
